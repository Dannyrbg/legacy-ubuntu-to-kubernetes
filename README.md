# legacy-ubuntu-to-kubernetes

This repository documents the migration of a legacy Java Spring Boot application
from an Ubuntu 18.04 virtual machine into a containerized Kubernetes platform
using Jenkins CI and AWS (ECR, EKS, RDS).
